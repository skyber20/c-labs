### Условие задачи
Составить программу на языке Си для построения и обработки дерева общего вида, содержащего узлы типа iпt. Основные функции работы с деревьями реализовать в виде отдельных
функций. После того, как дерево создано, его обработка должна производиться в режиме текстового меню со
следующими действиями:
- Добавление нового узла (для дерева общего вида должен задаваться родитель добавляемого узла. Добавляемый узел
становится самым младшим потомком);
- Текстовая визуализация дерева (значение каждого узла выводится в отдельной строке, с отступом,
пропорциональным глубине узла, в порядке старшинства узлов);
- Удаление узла (для дерева общего вида удаляется все поддерево, исходящее из удаляемого узла. Должно быть предусмотрено
корректное освобождение памяти);
- Определить глубину дерева. 

### Описание работы
Программа разделена на логические модули:
- **`tree.c`**: содержит реализацию дерева общего вида. Реализованы функции рекурсивного поиска узла, поиска родителя, вставки, удаления поддерева, нахождения глубины и визуализации.
- **`main.c`**: реализует текстовый интерфейс (CLI) для взаимодействия с пользователем.
- **`tests.c`**: содержит набор из 5 тестов с использованием `assert.h` для проверки корректности всех алгоритмов.

**Алгоритмы:**
- **Вставка**: программа находит узел-родитель по значению, переходит к его первому ребенку и по цепочке братьев находит последнего, к которому присоединяет новый узел*
- **Удаление**: сначала узел изолируется от основного дерева (перестраиваются связи родителя или левого брата), после чего рекурсивно освобождается память всего удаленного поддерева (снизу вверх)*
- **Глубина**: реализована рекурсивная функция, которая опрашивает всех детей узла, выбирает максимальную глубину среди их поддеревьев и прибавляет единицу.

### Запуск:
Для автоматизации сборки используется `Makefile`.

**Запуск основной программы (CLI):**
```bash
make main
```

**Запуск тестов**
```bash
make tests
```
